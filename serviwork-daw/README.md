# ServiWork

ServiWork és una plataforma digital d’intermediació de serveis que connecta **professionals** amb **clients (empreses o particulars)** de manera segura.

Aquest repositori conté el **prototip web** del projecte amb HTML, CSS i JavaScript, així com la documentació bàsica del disseny.

## Contingut del repositori

- `frontend/` → Carpeta amb les vistes, estils i scripts.
- `docs/` → Documentació, mockups i disseny de base de dades.
- `backend/` → Carpeta preparada per afegir futur backend (Laravel/PHP).
- `README.md` → Instruccions del projecte.

## Objectiu del projecte

Crear una plataforma on:
- Els professionals puguen oferir els seus serveis.
- Els clients puguen buscar, comparar i contractar.
- Es generen contractes simples i segurs.
- Es gestione informació de manera clara i legal.

## Tecnologies utilitzades

- HTML5
- CSS3
- JavaScript
